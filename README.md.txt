# Santa's Little Helper

Santa's Little Helper is a Chrome extension that makes web navigation effortless by allowing users to define custom shortcuts to their favorite websites. Type a keyword in the Chrome omnibox, and you're there!

## Features
- Custom URL shortcuts for quick access.
- Automatic fallback to Google search if no shortcut matches.
- Easy-to-use omnibox integration.

## Installation Instructions
1. **Download the Extension**: [Download the ZIP file](#) and extract it.
2. **Open Chrome Extensions Page**:
   - Go to `chrome://extensions/`.
   - Enable **Developer Mode** (toggle in the top-right corner).
3. **Load the Extension**:
   - Click **"Load unpacked"**.
   - Select the extracted folder `Santas_Little_Helper`.
4. **Start Using It**:
   - Type `h` in Chrome's address bar followed by a space.
   - Enter a shortcut (e.g., `yt` for YouTube) and press Enter!
